# rcore_addonaccount

optimization and replacement for "esx_addonaccount"

## Installation

1) Delete esx_addonaccount
2) Place rcore_addonaccount into resources folder
3) Start rcore_addonaccount instead of esx_addonaccount in your server.cfg
4) Enjoy :)
